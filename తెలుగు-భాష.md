# తెలుగు భాష

```text
01.   తేనె కన్న తీపి, తెలియగ పలికిన 
      జాతి కెల్ల వెలుగు, జాబిలమ్మ 
      అమ్మ పాల వంటి, అమృతపు సంపద
      జగతి లోన తెలుగు, జయము గొనును!
```
```text
02.   తల్లి భాష విడిచి, తరలిరి పరులకై
      పిల్ల లకును నేర్పు, పిచ్చి బాస
      తేనె వంటి తెలుగు, తీసి వేసిరి గద
      పరుల భాష మోజు, పాడు జేసె!
```
```text
03.   సగము తెలుగు మాట, సగము ఇంగ్లీషట
      కలిపి కొట్ట మనదు, ఘనత జూడు
      అచ్చ తెలుగు రాదు, ఆంగ్లమ్ము రాదయ్యె
      తెలుగు గాదు, అదియె తెంగిలీషు!
```
```text
04.   అమ్మ నుడిని విడచి, ఆంగ్లమ్ము వెదకుచు 
      "స్టైలు" కొరకు జనులు, సగము చెడిరి 
      తప్పు తడక లాడి, తలవంపు లొచ్చెను 
      తెలుగు బాస కిపుడు, తెగులు పట్టె!
```
```text
05.   హింది వారు గలయ, హిందీనె పలుకగ
      అరవ వారు గూడ, అరవ మాడు 
      తెలుగు వారు గలయ, తెగులుతో నాంగ్లమ్ము 
      సిగ్గు లేక మొరిగె, చిత్రముగను!
```
```text
06.   తేనె కన్న తీపి, తీయని మా భాష 
      రాగ యుక్త మైన, రమ్య భాష 
      సొగసు నింపు చుండు, సుకవుల కావ్యాల 
      దేశ భాష లందు, తెలుగు లెస్స!
```
